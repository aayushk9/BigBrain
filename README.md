# bigbrain

## what is bigbrain?
bigbrain is a web app that helps you discover and explore research papers on web3. you also get to read the latest crypto news from various sources. and yeah forgot to mention my favorite part about bigbrain, it also has a startups section where you can dive into everything about startups.

## why am i building this?
well I like to read research papers on web3, stay updated with crypto news and learn about startups. I couldn’t find a web app on the internet that provides all this in one place so I'm building one. 

## research paper browsing
- **smart search** – input specific queries and get **relevant papers**.  
- **curated knowledge** – access high-quality research papers without the hassle of manual searching.  

## tech stack (mern)
- **frontend**: react js
- **backend**: node.js + express  
- **database**: mongoDB
